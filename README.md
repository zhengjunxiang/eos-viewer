# EOS Browser

**使用webpack4 配置vue2 ssr 环境**

## Project setup
```
yarn install
or
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### pm2管理启动WEB server
> 需要先安装pm2: yarn global add pm2 Or npm install -g pm2
```
npm run pm2
```
